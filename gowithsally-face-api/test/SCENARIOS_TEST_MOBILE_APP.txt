COMMENT EXECUTER LES TESTS IA
================================================================================

1. Telecharger les images de test:
   cd gowithsally-face-api
   .\test\download_test_assets.ps1

2. Lancer les tests automatises:
   docker exec gws-face-api python /app/test/run_all_tests.py --verbose

3. Lancer un test specifique:
   docker exec gws-face-api python /app/test/run_all_tests.py -s detect
   docker exec gws-face-api python /app/test/run_all_tests.py -s gender
   docker exec gws-face-api python /app/test/run_all_tests.py -s age

4. Voir les resultats:
   Les resultats JSON sont dans test/results/
